#### JSBench.org Client
> JavaScript API client that connects to JSBench.org server to send benchmarks results
and receive total results.

**Goal**

The JavaScript client must be:

* stand-alone
* offline compatible
* simple (KISS)
* light-weight
* cross-browser compatible.

**Dependencies**

* Benchmark.JS
    * lodash
    * platform
* formToObject.js
* Handlebars
* Q promise library
